# RRTS — Rapid Rail Transit System UI Replica

**Live demo:** https://arpitgovil.github.io/RRTS/  
**Repository:** https://github.com/arpitgovil/RRTS

## Project overview
This project is a front-end replica of a Rapid Rail Transit System (RRTS) website — built to reproduce the layout, components, and visual styling of the referenced design. The site is implemented as a static, responsive frontend project and demonstrates component structure, layout techniques, and responsive styling.

## Features
- Static responsive UI for a transit/rail public-facing site
- Navigation, hero section, informational cards, and footer
- Modular source structure using TypeScript and CSS
- Live demo hosted on GitHub Pages

## Tech stack
- TypeScript
- Vite (development server & bundling)
- CSS for styling
- GitHub Pages for deployment

## Getting started (development)
1. Clone the repository:
   ```bash
   git clone https://github.com/arpitgovil/RRTS.git
   cd RRTS
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
   Check `vite.config.ts` / `package.json` for the build output path.
